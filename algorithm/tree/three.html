  1. 打家劫舍 III

小偷又发现了一个新的可行窃的地区。这个地区只有一个入口，我们称之为 root 。

除了 root 之外，每栋房子有且只有一个“父“房子与之相连。一番侦察之后，聪明的小偷意识到“这个地方的所有房屋的排列类似于一棵二叉树”。 如果 两个直接相连的房子在同一天晚上被打劫 ，房屋将自动报警。

给定二叉树的 root 。返回 在不触动警报的情况下 ，小偷能够盗取的最高金额 。

题目链接 ​

https://leetcode.cn/problems/house-robber-iii/description/

示例 ​

示例 1：

输入: root = [3,2,3,null,3,null,1]

输出: 7

解释: 小偷一晚能够盗取的最高金额 3 + 3 + 1 = 7

答案 ​

javascript
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @return {number}
 */
var rob = function(root) {
    // 分析 每一个点都有选中和不选中两种状态，采用map把每个点的两种状态存下来
    let checkMap = new Map()
    let noCheckMap = new Map()
    // 深度优先遍历树
    function traversal(node) {
        if(node === null) {return}
        traversal(node.left)
        traversal(node.right)
        // 当前点选中，则其左右子节点不选中
        checkMap.set(node, node.val + (noCheckMap.get(node.left) || 0) + (noCheckMap.get(node.right) || 0))
        // 当前点不选中，则其左右子节点选中和不选中都可以
        noCheckMap.set(node, Math.max(noCheckMap.get(node.left) || 0, checkMap.get(node.left) || 0) + Math.max(noCheckMap.get(node.right) || 0, checkMap.get(node.right) || 0))
    }
    traversal(root)
    return Math.max(noCheckMap.get(root) || 0, checkMap.get(root) || 0)
};

分析 ​

时间复杂度：O(n) 这个算法的时间复杂度是O(n)，空间复杂度是O(n)，其中n是二叉树的节点数。

空间复杂度：O(n)

这个算法的复杂度分析取决于递归过程中访问的节点数和存储中间结果的Map对象的大小。在最坏的情况下，如果二叉树是一个完全二叉树，那么它有n个节点，而高度h就是log(n)。因此，递归过程中访问的节点数就是n，而Map对象的大小也是n。因此，时间复杂度就是O(n)。

空间复杂度取决于Map对象的大小，而Map对象的大小就是n。因此，空间复杂度也是O(n)
